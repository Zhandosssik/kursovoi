# Курстық жұмыстарды тіркеу және бағалау жүйесі

Бұл жоба — студенттерге курстық жұмыстарын жүктеп, ал оқытушыларға оларды тексеріп, бағалауға мүмкіндік беретін толыққанды **Full Stack (PERN)** веб-қосымша. 

## 🚀 Қолданылған технологиялар (Tech Stack)

**Frontend:**
* React.js (Vite арқылы жинақталған)
* React Router DOM (Навигация)
* Axios (API сұраныстар)
* Context API (Қолданушы сессиясын басқару)

**Backend:**
* Node.js & Express.js
* PostgreSQL (Дерекқор)
* `pg` (PostgreSQL драйвері)
* JWT (JSON Web Token) + bcryptjs (Аутентификация және қауіпсіздік)
* Multer (Файлдарды жүктеу)

**Интеграция:**
* GitHub API (Репозиторий тілі мен жұлдыздар санын автоматты түрде алу)

## ✨ Негізгі мүмкіндіктері (Features)

### Студенттер үшін:
- Жүйеге тіркелу және кіру (JWT авторизация).
- Курстық жобаның тақырыбын және сипаттамасын қалдыру.
- Жұмыс файлдарын (PDF, DOCX) серверге жүктеу.
- Жобаға ашық GitHub репозиторийінің сілтемесін қосу.
- Өз жұмысының мәртебесін көру (Күтуде / Қабылданды / Қайтарылды).

### Оқытушылар үшін:
- Барлық студенттердің жүктеген жұмыстарын көру.
- Студенттердің файлдарын жүктеп алып, оқу.
- Жұмысты бағалау (0-100), пікір қалдыру және мәртебесін өзгерту.

## 🛠 Жобаны іске қосу нұсқаулығы (How to Run)

Жобаны локалды компьютерде іске қосу үшін келесі қадамдарды орындаңыз:

### 1. Дерекқорды орнату (PostgreSQL)
`server/src/config/db.js` ішіндегі дерекқор мәліметтерін (пароль, порт) өз компьютеріңізге сәйкестендіріңіз. Содан кейін PostgreSQL-де қажетті кестелерді құрыңыз (SQL кодтары қолжетімді).

### 2. Backend (Сервер) іске қосу
```bash
cd server
npm install
npm run dev
